增长黑客：如何低成本实现爆发式成长

更新于2022-11-17|Learn
|字数总计:7.6k|阅读时长:22分钟|阅读量:

脸谱网如何从默默无闻到坐拥二十几亿用户？爱彼迎、优步何以在短短时间估值超过百亿美元？领英怎样跨步成为全球领先的职业社交平台？

这些初创公司实现爆发式成长的共同奥秘就是增长黑客。增长黑客是硅谷当下热门的新商业方法论，其精髓在于通过快节奏测试和迭代，以极低甚至零成本获取并留存用户。

作为最早提出 “ 增长黑客 ” 概念的理论先驱、带领 Dropbox 实现 500% 增长的实战领军人物，作者在书中分享了如何跨部门搭建增长黑客团队，以及实现用户和利润双增长的具体行动指南。

内法

搭建增长团队

1、最有可能成功的增长策略不是只关注用户群的扩大，也要最大限度地发挥现有用户的价值。

2、跨职能团队不仅能够促进并加速产品、工程、数据和市场部门之间的合作，并且能够激励团队成员更多地了解并理解其他成员的视角及他们的工作。

3、增长团队由增长负责人、产品经理、软件工程师、营销专员、数据分析师、产品设计师构成，其构成和职责（包括团队的规模、组织结构和具体的任务和重心）必须符合公司的需要。

4、增长黑客过程是一个持续的循环，其步骤是数据分析与洞察收集 → 想法产生 → 排定试验优先级 → 试验执行。通过循环往复地推进这一过程，增长团队将不断积累大大小小的成功，创造一个不断改进的良性循环。

5、公司的组织汇报结构必须对增长团队有清晰的界定，即增长负责人应向谁汇报，增长团队有两种常见的汇报结构。

特点
职能模式（或产品部门主导模式） 增长团队向主管其工作范围内的某个产品或几个产品的高管汇报，适用于成熟的企业或已处于创业后期的公司
创建一个独立的团队 该团队不归属于既有的产品开发团队，增长团队负责人向增长副总裁汇报，而增长副总裁一般直接向 CEO 或其他高管汇报

好产品是根本

1、好产品是增长的根本，增长团队必须要了解产品的核心价值是什么、对哪些客户来说具备这些核心价值以及为什么。

2、啊哈时刻是产品使用户眼前一亮的时刻，是用户真正发现产品核心价值——产品为何存在、他们为何需要它以及他们能从中得到什么——的时刻，即用户认识到这个产品对他们来说为什么不可或缺。

3、评估不可或缺性的第一种方法是肖恩调查，它用于确定产品是否满足基本要求，确定产品是否已具备核心价值，调查的目标群体最好是活跃用户而不是休眠用户，其具体内容如下。

4、评估不可或缺性的第二种方法是看用户留存率，即在一定时间内继续使用产品或付费使用产品的用户比例，不同类型的业务或产品有不同的留存率，最好能够以行业内有足够可比性的成功产品作为基准。

5、当产品评估为未实现不可或缺性时，增长团队必须采用分析手段找出啊哈时刻没有实现的原因以及如何才能实现，而不是单纯依靠主观臆测，同时采取三个措施，一是开展更多的客户调查，包括采访和实地调研，真正与客户和潜在客户交谈；二是针对产品改动和信息传达进行高效试验；三是深入分析用户数据。

6、成本低、易使用的分析工具和网络营销技术的发展使产品和信息传达测试变得非常容易，通过这样的测试可找到客户群和功能的完美组合，打造不可或缺的产品，而大多数增长团队会采用最小可行测试的做法。

7、A / B 测试是以低成本甚至零成本很快改变信息传达方式并加以测试，它在两个及两个以上随机选择的群组中测试两种不同的信息传达方式，以此确定哪一种能够获得更优的用户反应，同时数据分析必须能够在任何 A / B 测试中跟踪测试对象从点击到长期使用整个过程的行为，以提高测试的精准程度。

8、针对实体产品的试验的方法还包括构建最简单的产品原型然后让用户试用、制作视频或计算机演示以展示产品的新功能，同时观察客户的反应，团队应对耗时耗力的测试通过严密的论证将风险降到最低，且在开展规模更大、风险更高的试验的同时运行一些更稳健的测试，从而使产品获得可持续的增长。

9、营销人员、数据科学家和工程师必须通力合作，在网站、移动应用、销售点系统、邮件营销和客户数据库中加入合适的数据跟踪设置，对不同来源的用户信息加以整合，以获得全面、充分的用户行为数据，以便数据团队进行分析，从而构建一个数据湖泊或数据仓库，即储存所有客户信息的独特应用，可真正深入挖掘并发现可能对产品的使用方式不同于其他人的用户群，从单个用户的层面探索产品的使用情况。

10、收集并分析数据的第一步是通过事件跟踪来挖掘用户或客户的关键行为，同样增长团队必须跟踪客户在整个客户体验过程中的各种活动，即他们从访客成为新客户，再从新客户成为固定忠实客户的过程中的各个事件。该阶段的主要任务是寻找那些认为产品不可或缺的客户，即频重新定位产品繁使用或购买产品的人，并发现他们不同于其他客户之处。

11、所有这些试验和分析过程都应聚焦于发现所提供给客户或能够提供给客户的啊哈时刻，一旦确定创造啊哈体验的条件，增长团队就应将注意力转移到如何尽快使更多客户体验到这一时刻。

确定北极星指标

1、创造并使更多用户体验到啊哈时刻是破解增长难题的第一步，而下一步是明确增长战略，增长团队必须在快节奏增长试验前清楚地知道将如何驱动增长——增长杠杆是什么以及它们是否能够帮助取得理想的结果。

2、确定增长战略和增长重点的第一步是明确哪些指标对产品增长最为重要，最佳方式是建立基本增长等式，所有与增长相关的关键因素都在这个等式中有所体现，而这些因素相加共同驱动公司的增长。即这个等式是核心增长杠杆的集合，而每个产品或企业的等式都不尽相同。

3、北极星指标应能够最为精准地抓住企业为客户创造的核心价值，要反复问自己增长公式中的哪一个变量最能反映产品不可或缺体验的实现情况，以完善增长等式、缩小关注范围，从而选择一个关键的能够决定最终成败的指标来指导所有的增长活动，但北极星指标并非一成不变，而是会随着企业的增长而发生变化。

4、确定增长等式并明确北极星指标的前提是能够收集客户行为数据并衡量产品表现和试验结果，整合所有的数据资源对客户整个体验过程进行详细的跟踪，只有这样才能知道对产品表现和试验结果的假设是否符合用户使用产品的真实情况，但数据不是唯一，定量分析的同时也要展开定性分析，如用户反馈，以拓展试验思路。

5、数据分析及试验结果要以简洁明了的报告进行输出与汇报，而总览图用于缩小汇报范围，聚焦关键趋势或指标，其指标常以比例而非静态数据的形式呈现，或将指标数据与既定目标进行比较，从而帮助团队成员掌握业务增长情况并根据总览图提供的信息对相关问题给出快速清晰的回答。

6、确定增长等式与关键指标、恰当的数据跟踪设置、数据采集和包含客户反馈的报告能够帮助团队明确并监测核心增长杠杆的变化，这是增长黑客过程走向成功的关键而有力的第一步，而增长黑客过程的具体步骤是提出最值得测试的一组想法 → 以最为高效的方式开展试验 → 召开流程严密的增长会议 → 持续从试验结果中吸取经验教训并进一步巩固试验成果，以加快增长引擎的运转速度。

快节奏试验

增长黑客循环分为四个阶段，即分析数据并收集洞察，形成试验想法，排定试验优先级，运行试验，再回到分析阶段审视试验结果并决定下一步行动，且每一个步骤的完成都应当保持固定的节奏，以每一两周完成一次循环为佳，增长团队每周应召开一次一小时左右的会议，审视试验结果并决定下一周要执行的试验。

外战

获客

1、不要在实现产品—市场匹配前火力全开采取大规模的获客行动，但对具有网络效应的公司来说，获取用户的行动通常必须与产品开发同步进行。扩大获客规模首先要实现另外两种匹配，一是语言—市场匹配，即对产品优势的描述打动目标用户的程度；二是渠道—产品匹配，即所选择的营销渠道在向目标用户推广产品时的有效程度。

2、语言—市场匹配用于衡量描述和推广产品的语言能在多大程度上打动潜在用户，促使他们试用产品，涵盖营销活动所有环节中使用的语言，包括电子邮件、移动推送以及印刷和网络广告等。设计的语言必须非常简洁地传达出产品的核心价值（啊哈时刻）且回答每位消费者最关心的简单问题 “ 你展示的这个产品将如何改善我的生活？”。

3、增长黑客试验可相对快速地发现一两个最优渠道，甚至抢先于竞争对手找到它们，增长团队在发现阶段应广泛试验各种可能的渠道选项，对各种渠道进行深入调研，最后选出若干优先项进行试验，一旦找到一两种高度匹配的渠道，就可进入第二个阶段。增长团队在优化阶段应在扩大规模的同时努力实现渠道成本效益比和影响力的最大化。

6、病毒式增长分为口碑式病毒和原生病毒（内置于产品），创造原生病毒必须遵循产品开发的基本原则，即让分享产品这种体验变得不可或缺，或让这种体验尽可能便捷、舒心。任何产品的病毒性由有效载荷（每位用户每次向多少人发送广告）、转化率（邀请转化率）和频率（人们收到邀请的频率）三个因素决定，即病毒性 = 有效载荷 × 转化率 × 频率。

7、具有网络效应的产品在获得病毒式增长方面具有得天独厚的优势，使用产品的人数越多产品体验就越好，人们越愿意鼓励其他人加入。如果用户体验本身并不对用户分享产生激励，可能就需要创造激励措施，通常的做法是提供某种奖励，但关键是用户完成推荐后得到的奖励必须与产品的核心价值相关。

激活

1、提高激活率的核心在于让新用户更快地体验到啊哈时刻，感受到产品不可或缺的人越多，忠于产品的用户就越多。

2、破解激活的第一步是找出通往啊哈时刻的所有节点，列出新用户在体验到啊哈时刻前必须完成的所有步骤。一旦确定通往啊哈时刻的所有步骤，接下来寻找障碍，找到用户逃离原因的第二步就是计算每个步骤的转化率，即计算整个用户旅程中完成每个步骤的用户比例。

3、测量转化率的最佳途径之一是创建漏斗报告，即显示抵达用户旅程中每个关键步骤的用户比例，同时它也显示每个环节的流失率。另外，报告还应跟踪访客接触产品的途径或渠道，总结每种渠道中阻碍用户迈向下一步的因素。

4、针对流失率很高的环节依然继续使用产品的用户以及弃用产品的用户进行问卷调查和采访，找出流失原因，问卷调查必须非常简短且发出时必须满足两个主要条件，一是用户活动反映出用户的困惑，如某个页面停留时间过长或离开某个页面；二是用户刚刚完成很多人没有完成的步骤，如创建账户或点击购买。要想知道用户为何会在某个节点放弃购买，最有用的回答往往来自那些没有放弃购买的用户。

5、用户体验设计的摩擦是指阻碍人们完成他们想完成的行动的令人心烦的障碍，漏斗转化报告可看作是寻找用户旅程中摩擦源的路线图。设计并优化新用户体验的首要原则是将其视为用户与产品的一次独一无二的邂逅；第二个原则是新用户体验的第一个着陆页必须完成三个根本任务，即传达相关性、展示产品价值和提供明确的行为召唤。

6、单点登录和翻转漏斗可十分有效地消除摩擦且广泛适用于各种业务和产品。单点登录或社交账号登录是指只需点击鼠标就能创建新账户，减少人们必须提供的个人信息可大幅提高注册量，从而降低注册过程中的摩擦；翻转漏斗是指在邀请访客注册前先让他们体验产品带来的乐趣，从而减少阻碍用户迅速体验啊哈时刻的摩擦。

7、提升新用户激活率时可能需要靠增加阻力来引导用户进入后续环节，一方面要引导用户完成整个过程并最终实现激活，另一方面又不能给用户带来过多负担，令他们望而却步。优化激活中最看似矛盾的一点在于并非所有的摩擦都带来负面影响，有时需要设置一些积极的摩擦，如果操作得当，用奖励的方式通过行动逐渐提高用户承诺，摩擦也可成为激活与增长的催化剂。

8、学习流不仅可促使人们注册，还能有效教育新用户，使他们了解产品优势、价值及使用方法，这种新用户体验设计利用新用户在初次使用产品时给予你的大量关注和耐心，能确保用户在体验结束时就可以被激活。

9、问卷调查和新用户体验游戏化是非常有效的积极摩擦策略。问卷调查是指在欢迎用户的同时向他们提一些问题，所提的问题应明确围绕如何更好地服务用户，询问用户有哪些兴趣或他们希望解决哪些问题，不仅能创造一种承诺，还能在用户与产品间也建立起更深层次的关系。游戏化本质是一种奖励机制，如给采取某个行动的用户提供其他人无法获得的奖励和好处，游戏化要满足有意义的奖励，通过改变赢取和展示奖励的方法来创造惊喜和乐趣，以及提供能带来即刻满足感的要素，最有效的奖励包括地位、权限、权力和实物（指金钱奖励或礼品）。

10、游戏化做法并非对所有情况都适用，但触发物无处不在，触发物是指任何刺激人们采取行动的提示，包括邮件通知、移动推送、着陆页上的行为召唤。触发物的力量来源于用户动机的强弱以及采取行动的能力两个关键因素，即它们能在多大程度上激发用户采取希望他们采取的行动，以及用户收到触发物后付诸行动的难度。

11、根据用户动机和能力水平可将触发物分为三个基本类型，协助型触发物用于帮助那些动机很强但能力很低的用户采取行动；信号型触发物用于帮助那些动机和能力都很强的用户沿着正确方向前进并鼓励他们重复行动；刺激型触发物用于刺激那些能力很强但动机不足的用户采取行动。

留存

1、提高留存必须不断设计和测试增长手段，任何公司都应努力增加产品给用户带来的价值并逐渐提高用户参与度。

2、留存分为初期、中期和长期三个阶段，不同的留存阶段采用不同的留存方法。初期阶段决定用户是继续使用或购买产品或服务还是使用一两次后就沉睡，其时间长度应根据产品的行业标准以及对用户行为的分析来决定；中期阶段的核心任务是让使用产品成为一种习惯，让用户逐渐从产品或服务中获得满足感；长期阶段要确保产品继续为用户带来更大价值，必须试验各种方法不断完善产品，其关键在于让用户不断重新认识到产品的不可或缺性。

3、不同的公司衡量留存的方法可能不同，用户购买产品或使用服务的频率很大程度取决于产品或服务本身的性质。设计留存指标要将计算结果与市场调研得出的同类产品或服务的最高留存率进行比较，或与成功的竞争对手进行比较。

4、利用群组分析方法可确定不同用户群的留存率，从而更加深入地解读数据，找出用户留下或离开的原因，最基本的做法是按照获客时间进行划分，即按用户完成注册或第一次购买的日期来分类。

5、改善初期留存的增长手段与改善激活的增长手段本质一样，前者是后者的一种延伸。不断优化新用户体验，让用户尽可能快地体验到产品核心价值是这一阶段最重要的两大策略；使用移动推送和邮件等触发物也能够有效强化用户对产品实用性和价值的认识，但增长团队在留存初期不应只想着靠触发物来唤回用户，还应关注产品的用户体验以及改进产品。

6、留存中期的核心目标是让用户养成使用产品的习惯，从而巩固用户对产品的忠诚度，其关键在于让用户确信继续使用产品或服务能带给他们持久的回报，增长团队应找出触发物的最佳使用数量、方法和频率，以创造并强化用户习惯。

7、增长团队不应把目光局限在金钱或省钱的回报，而应创造性地思考可提供什么样的无形回报，而且还应试验将实际回报和体验式的社交回报结合起来，如社交回报（点赞）、飞行常客奖励计划，同时回报应契合产品价值。

特点
品牌大使计划 通过授予用户高级用户的称号来给用户提供社会认可以及其他的实际好处。
认可用户成就 肯定用户的某种成就或行为，或使用行为邮件（用户完成里程碑事件即发送祝贺邮件）。
客户关系个性化 个性化一对一体验。

8、A / B 试验可帮助团队确定公布消息的最佳时机，一旦在一定规模的用户群中实现强劲的留存，下一步就是专注于如何让用户乐于使用产品并长期保持高度活跃，一要优化现有的产品功能、推送以及对重复使用的奖励；二要在一个较长的时间周期里定期推出新功能，把握好这二者间的平衡极其重要。

9、长期留存的另一个关键要素是弄清如何让用户沿着学习曲线前行，即持续的用户引导——首先完成容易实现的小目标，然后逐渐提高熟练程度，应将这种学习进程嵌入任何需要用户掌握的新功能中，让他们从产品中获得最大价值。增长团队应测试持续的用户引导中所有的信息传达，如向试验组发送有关新功能使用的不同版本的文字和图片说明。

10、复活僵尸用户的第一步是调查用户当初消失的原因，并通过发送电子邮件或定向广告，提醒他们产品的啊哈时刻或当初吸引他们的产品核心价值，以此来挽回用户。增长团队要试验信息传达的措辞、频率以及持续时间来确保请求用户回来的信息不会惹烦或进一步疏远用户，否则只会让他们对产品评价更差，从而彻底击碎挽回他们的一切希望。

变现

1、破解变现的第一步是分析数据，找出最具潜力的试验，增长团队要回到根据整个用户旅程绘制的基本路线图，在路线图上标出从获客到留存的整个过程中所有可能从用户身上盈利的机会。

2、增长团队要根据用户创造收益的高低、用户参与度（尤其是用户对广告感兴趣的程度）等指标来建立和划分群组，包括但不限于地点、年龄和性别、客户购买商品的类型或使用的功能、获客渠道、使用的访问设备、使用的网络浏览器、某个时间段内访问网站或 APP 的次数、用户第一次购买或第一次采取行动的日期等。此阶段的重点不是寻找留存规律，而是寻找每个群组和收益之间的关联，从而帮助团队找到试验想法。

3、增长团队要按照某些相同特征对用户进行大致分类，这些特征可能包括用户来自同一地区，有着相同的经历，消费水平大致相同，有相同的产品需求或几个因素的组合，创建分组的目的在于更好地想出满足客户特殊需求的方式。

4、增长团队要通过调查问卷直接向用户了解每个关键群组最希望看到的产品改进，增加营收的核心在于向用户提供让他们觉得很有吸引力而且最能满足他们需求的产品和服务。

5、增长团队要利用数据和算法为用户定制产品和功能，以建立更加牢固的客户关系，从而留住客户。杰卡德系数公式通过计算两个产品之间的相似度，从而向用户推荐他们可能购买的额外商品，其计算结果显示出人们通常同时购买这两样商品，其公式是 J（A，B）＝（∣A∩B∣）/（∣A∪B∣）＝（∣A∩B∣）/（∣A∣－∣B∣－∣A∩B∣）J（A，B）＝（|A∩B|）/（|A∪B|）＝（|A∩B|）/（|A|－|B|－|A∩B|）J（A，B）＝（∣A∩B∣）/（∣A∪B∣）＝（∣A∩B∣）/（∣A∣－∣B∣－∣A∩B∣）。

6、测试个性化定制策略可以先向一小部分用户发送个性化邮件和短信推送，且评估这些用户的反应和个性化定制的效果，完成这一步后增长团队就能获得用户体验有效性方面的早期数据，如果结果积极，团队就可进一步在个性化方面做投资。

7、增长团队可与产品和财务团队合作，通过开展问卷调查和用户调研找出最佳定价区间并进行试验。

8、定价相对论是指人们对于价格高低的判断会受到其他价格选项的影响，增长团队应试验增加一个产品选项，利用定价来帮助用户更好地了解向他们出售的产品和计划的相对价值。

9、消费者心理学对变现用户、增加营收有很大的价值，罗伯特·西奥迪尼提出的影响用户行为的六原则包括互惠原则、承诺和一致原则、社会认同原则、权威原则、喜好原则、稀缺原则。
